# Express + MongoDB with Docker & Docker Compose

This repository is for practicing Docker and Docker Compose by running:
- An Express (Node.js) API
- A MongoDB database
- A persistent Docker volume for MongoDB data

The Express app connects to MongoDB using a connection string provided via environment variables.

---

## Requirements

- Docker Engine
- Docker Compose (Compose plugin)

Check versions:

```bash
docker --version
docker compose version
